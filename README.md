# NumCompute

# Rough Draft documentation for contribution:
1. Clone the repo 
2. Go to the project's directory 
3. Run command `pip install .`
4. Make your changes - adding/editing modules
5. Write tests in tests folder, file name format   `test_<module>.py`
6. Run command `pip install .`
7. To run test run command `python -m pytest <file name>` in tests folder. 
8. Upadate the .gitignore, if needed, before pushing code. We don't have to push build and test artifacts.
